## The CodeBird
### Note 
Please install ```pnpm``` in your Global Package 
Do Not Push Directly In The **main** Branch. Push It in **dev** Branch

## Installetion

```
 npm install -g pnpm
```

### Install The Packages 

```
pnpm install
```

### Run The Web

```
pnpm dev
```
